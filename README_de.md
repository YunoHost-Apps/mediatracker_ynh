## Übersicht

Self hosted platform for tracking movies, tv shows, video games, books and audiobooks, highly inspired by flox

### Features

- notifications
- calendar
- multiple users
- REST API
- watchlist
- docker image
- import from Trakt
- import from goodreads


**Ausgelieferte Version:** 0.2.11~ynh2
